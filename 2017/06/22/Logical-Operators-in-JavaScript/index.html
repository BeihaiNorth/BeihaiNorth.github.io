Logical Operators in JavaScript

2017-06-22
JavaScript

This artical tells the basic use and smart practice of three logical operators in JavaScript.

Description

Unlike in Java or C++, logical operators here can be used with non-bool type. If a value can be converted to true, the value is so-called truthy. If a value can be converted to false, the value is so-called falsy. As logical expressions are evaluated left to right, they are tested using short-circuit evaluation:

Expression Returns
falsy && expr false
truthy && expr expr
falsy || expr expr
truthy || expr true
!expr false if expr can be converted to true; otherwise, true.

Expressions that can be converted to false are:

  • NaN;
  • null;
  • 0;
  • undefined;
  • empty string("");
